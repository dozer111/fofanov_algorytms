Сортировка Шелла(shellSort)

Улучшение сортировки выборками

Зная суть простых алгоритмов сортировки, можно сделать свой алгоритм, основывающийся на них
Сортировка была придумана в 1959году Р.Шеллом

Главная фишка - InsertingSort крайне плох, если одно из самых маленьких значений находится в конце, но стремится к 
линейной сложности, если массив почти отсортирован
Значит, давайте ка сначала предварительно отсортируем массив, а затем предварительные данные прогоним 
через InsertionSort

Предварительная сортировка происходит с помощью GAP
GAP - целочисленное значение шага между елементами

Сортировка начинается с большого GAP, и заканчивается постепенным уменьшением GAP вплоть до GAP == 1

Стандартная схема работы с GAP
1. высчитать максимальный(начальный GAP)
int gap = 1;
while(gap < array.Length < 3)
{
	gap = 3 * gap + 1;
	// gap - значение k с уравнения ((3^k)-1)2
}






2. ПРЕсортировка с постепенным уменьшением GAP

while(gap >=1)
{
	// sort ... 
	// sort ... 
	// sort ... 
	// sort ... 
	gap /=3;
}

Стандартная формула последовательности значений GAP-ов: ((3^k)-1)2
